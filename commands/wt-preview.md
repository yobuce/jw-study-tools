# 파수대 연구 예습자료

특정 주의 **파수대 연구 기사**를 항별로 구조화해(질문·성구·각주·참조 출판물) **예습용 자료**로 정리합니다. 해설 스크립트는 만들지 않고, 스스로 예습할 수 있는 자료를 표/문서로 제공합니다.

## 입력

$ARGUMENTS (연구 주간 날짜 또는 파수대 식별자. 예: `20260615`, "6월 15일", "이번주", `w_KO_202604`. 비어 있으면 오늘이 속한 주)

## 실행 절차

### 1단계: 대상 파수대 파일 찾기

파수대 연구는 보통 발행 약 2~3개월 뒤에 진행한다. 데이터 폴더 결정 후 후보 파일을 `--list` 로 확인해 입력 주간을 포함한 기사를 찾는다.

```bash
JW_DATA_DIR="${user_config.data_dir}"
[ -z "$JW_DATA_DIR" ] && JW_DATA_DIR="${CLAUDE_PLUGIN_DATA}/JWPUB"
export JW_DATA_DIR
# 식별자를 직접 받았으면 그대로 사용. 날짜만 받았으면 (대상월-2),(대상월-3)의 w_KO 파일을 --list 로 점검
python -X utf8 "${CLAUDE_PLUGIN_ROOT}/scripts/extract_wt_article.py" w_KO_YYYYMM --list
```

`--list` 출력의 `[YYYY.MM.DD~MM.DD]` 연구 주간 범위에서 입력 날짜가 들어가는 기사 번호 N을 고른다. 어떤 파일에 있는지 모호하면 후보 파일 여러 개에 `--list` 를 돌려 맞는 주간을 찾는다.

### 2단계: 기사 구조화 추출

```bash
python -X utf8 "${CLAUDE_PLUGIN_ROOT}/scripts/extract_wt_article.py" w_KO_YYYYMM --article N
```

- 날짜로 바로 찾으려면 `--article N` 대신 `--date YYYYMMDD` 사용 가능
- 기계 가공이 필요하면 `--json` 으로 구조화 출력

추출 결과에는 항별 **질문, 인용 성구, 각주, 참조 출판물**이 포함된다.

### 3단계: (선택) 성구 연구 자료 보강

핵심 성구의 연구 노트·각주·연구 자료 찾아보기를 덧붙여 예습 깊이를 더한다:

```bash
python -X utf8 "${CLAUDE_PLUGIN_ROOT}/scripts/lookup_bible.py" "<성구1>; <성구2>" --study
```

### 4단계: 예습자료 출력

다음 형태로 정리한다(기본은 항별 표). **해설/답안을 대신 작성하지 말 것** — 스스로 채우도록 돕는 자료를 만든다.

- **항별 표**: 항 | 질문 | 핵심 성구 | 참조(각주·출판물)
- **예습 문서**: 항별로 ① 질문 ② 관련 성구(필요 시 본문 인용) ③ 참고할 연구 자료 포인트를 정리하고, 답은 비워 둔다
- **md 파일 저장**: 사용자가 원하면 현재 작업 폴더에 `파수대예습_YYYYMMDD.md` 로 저장

먼저 항별 표로 개요를 보여 준 뒤, 더 자세한 예습 문서나 파일 저장이 필요한지 사용자에게 물어 다듬는다.

### 5단계: 용어 검토

정리·안내 문구 등 **내가 직접 쓴 한국어 문장**에 대해 `${CLAUDE_PLUGIN_ROOT}/docs/용어가이드.md` 의 규칙을 적용해 점검한다. **항 질문·성구·각주 등 추출·인용한 원문은 고치지 않는다.**

## 데이터 요구

`$JW_DATA_DIR/파수대/w_KO_YYYYMM.jwpub` 필요. 성구 본문·연구 자료를 함께 보려면 `$JW_DATA_DIR/nwtsty_KO.db`, `$JW_DATA_DIR/rsg19_KO.jwpub` 도 필요합니다.
