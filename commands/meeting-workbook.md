# 집회교재(주중집회) 자료 정리

집회교재(mwb)에서 특정 주의 **주중집회 전체 순서**와 각 부분의 **참조 출판물·성구**를 추출해 보기 좋게 정리합니다. 원하면 성구의 연구 자료 찾아보기까지 보강해 문서로 저장합니다.

## 입력

$ARGUMENTS (날짜. 예: `20260615`, "6월 15일", "이번주". 비어 있으면 오늘이 속한 주로 간주)

## 실행 절차

### 1단계: 날짜 → 집회교재 파일 결정

1. `$ARGUMENTS`에서 날짜를 파싱한다(비어 있으면 오늘 기준 이번 주).
2. 집회교재는 격월 발행이므로 해당 월로 파일을 정한다:
   - 1~2월 → `mwb_KO_YYYY01` / 3~4월 → `mwb_KO_YYYY03` / 5~6월 → `mwb_KO_YYYY05`
   - 7~8월 → `mwb_KO_YYYY07` / 9~10월 → `mwb_KO_YYYY09` / 11~12월 → `mwb_KO_YYYY11`
3. 데이터 폴더 결정 후 파일 경로는 `$JW_DATA_DIR/집회교재/mwb_KO_YYYYMM.jwpub`.

### 2단계: 교재 추출

```bash
JW_DATA_DIR="${user_config.data_dir}"
[ -z "$JW_DATA_DIR" ] && JW_DATA_DIR="${CLAUDE_PLUGIN_DATA}/JWPUB"
export JW_DATA_DIR
python -X utf8 "${CLAUDE_PLUGIN_ROOT}/scripts/read_jwpub.py" "$JW_DATA_DIR/집회교재/mwb_KO_YYYYMM.jwpub" --refs
```

- `--refs` 는 본문 + 성구 참조 + 참조 출판물을 함께 출력한다(순서·참조 정리에 적합).
- 더 깊은 연구 자료 찾아보기까지 필요하면 옵션 없이(전체 추출) 다시 실행한다.
- 어떤 주가 들어 있는지 모르면 `--text-only` 로 먼저 훑어 입력 날짜에 해당하는 주간을 찾는다.

### 3단계: 해당 주 순서 정리

추출 결과에서 입력 날짜의 주간을 찾아 **전체 순서**를 정리한다:

1. **여는 말 / 노래·기도**
2. **하느님의 말씀에서 보화 찾기** — 보화 강연 제목, 영적 보물 찾기, 성경 낭독 범위
3. **전도 활동에 능숙해지기** — 각 과제(시간·유형·장면)
4. **그리스도인으로 살기** — 생활 부분(들), 회중 성서 연구 범위
5. **마치는 말 / 노래·기도**

각 부분마다 **참조 출판물**(예: 「파」, 「와」, 「훈」 등)과 **등장 성구**를 함께 모은다.

### 4단계: (선택) 성구 연구 자료 보강

사용자가 원하면 등장 성구들에 대해 연구 자료 찾아보기를 보강한다:

```bash
python -X utf8 "${CLAUDE_PLUGIN_ROOT}/scripts/lookup_bible.py" "<성구1>; <성구2>" --study
```

### 5단계: 출력 형태

사용자 요청에 맞춰 다음 중에서 정리한다(기본은 표):

- **표**: 부분 | 시간 | 제목/범위 | 참조 출판물 | 성구
- **순서 문서**: 위 5단계 순서를 글머리표로 정리
- **md 파일 저장**: 사용자가 원하면 현재 작업 폴더에 `집회교재_YYYYMMDD.md` 로 저장

먼저 표로 보여 준 뒤, 더 자세히/문서로/파일로 원하는지 사용자에게 물어 다듬는다.

### 6단계: 용어 검토

요약·표 설명·안내 문구 등 **내가 직접 쓴 한국어 문장**에 대해 `${CLAUDE_PLUGIN_ROOT}/docs/용어가이드.md` 의 규칙을 적용해 점검한다(통치체→중앙장로회, 형제님→형제, 사역→봉사, 교회→회중 등). **추출·인용한 원문은 고치지 않는다.**

## 데이터 요구

`$JW_DATA_DIR/집회교재/mwb_KO_YYYYMM.jwpub` 필요. 성구 본문·연구 자료를 함께 보려면 `$JW_DATA_DIR/nwtsty_KO.db`, `$JW_DATA_DIR/rsg19_KO.jwpub` 도 필요합니다.
