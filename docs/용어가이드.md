# 용어 가이드 (jw-study-tools)

이 플러그인은 JWPUB·자막을 **추출·정리**하는 도구다. 출력물은 대부분 ① 출판물에서 그대로 가져온 **원문(인용)** 과 ② 에이전트가 직접 쓴 **요약·표·정리 문장**으로 나뉜다. 두 가지를 다르게 다룬다.

## 핵심 원칙: 원문은 보존, 교정은 내가 쓴 문장에만

- **추출·인용한 원문은 절대 고치지 않는다.** 성구 본문, 연구 노트·각주, 파수대 항 질문, 집회교재 문구, 동영상 자막 등은 원문 그대로 보여 준다. (출판물 본문에 "예수님", "통치체" 등이 있어도 그대로 둔다.)
- **교정 대상은 에이전트가 직접 작성하는 부분뿐이다.** 집회 순서 요약, 표의 설명란, 예습자료의 안내 문구, 정리·해설성 문장 등 *내가 새로 쓰는 한국어*를 아래 규칙에 맞게 쓴다.
- 따라서 `/lookup-verse`, `/lookup-study`, `/read-jwpub`, `/extract-jwb-subtitle` 처럼 **원문을 그대로 내보내는** 명령에서는 교정이 거의 필요 없다. `/meeting-workbook`, `/wt-preview` 처럼 **내가 요약·정리 문장을 덧붙이는** 명령에서 이 규칙을 적용한다.

## 교정 규칙 (내가 쓰는 문장에 적용)

| 잘못된 표현 | 올바른 표현 | 비고 |
|---|---|---|
| 통치체 | 중앙장로회 | |
| 통치체 성원 | 중앙장로회 성원 | |
| 제명 / 제명되다 / 제명된 사람 | 내보내진 사람 / 내보내지다 | 명사형은 "내보내진 사람", 동사형은 "내보내지다" |
| 형제님 | 형제 | "님" 호칭 금지 |
| 자매님 | 자매 | "님" 호칭 금지 |
| 형제님께서 ~하셨습니다 | 형제가 ~했습니다 | 주어 "형제/자매" 뒤 술어에 주체높임 '-시-' 결합 금지 |
| 자매님께서 말씀해 주셨습니다 | 자매가 말했습니다 | 동일 ("말씀하셨다" → "말했다") |
| 그 형제분이 오셨습니다 | 그 형제가 왔습니다 | "형제분/자매분"도 금지 |
| 장로님 | 장로 | "님" 호칭 금지 |
| 장로 형제 | 형제 | **직책을 반드시 지칭해야 하는 경우가 아니라면** "장로 형제"도 쓰지 않고 그냥 "형제"로. 직책 지칭이 필수일 때만 "장로"로 표기 |
| 사역 / 사역하다 / 사역자 | 봉사 / 봉사하다 / 봉사자 | "사역" 일괄 금지 |
| 목회 / 목회하다 / 목회자 | (대체어 없음 — 문맥에 맞게 재작성) | "목회" 일괄 금지. 의미상 가까운 활동은 "봉사", "양 떼를 돌보는 일", "장로의 직무" 등으로 풀어 쓴다 |
| 교회 (여호와의 증인을 지칭) | 회중 | **개신교·가톨릭의 교회**를 가리키는 경우에만 "교회" 사용 가능. 여호와의 증인 모임/단체에는 절대 "교회"라고 쓰지 않고 "회중"으로 표기 |
| 간증 / 간증하다 | 경험 / 경험담 (또는 그에 맞는 동사) | "간증" 일괄 금지. 굳이 명사가 필요하면 "경험", "경험담" 사용 |
| 예수님 | 예수 | "님" 호칭 금지. "예수께서", "예수께서 말씀하셨습니다"처럼 조사·어미만으로 존대 표현. 단, **인용 성구·원문에 "예수님"이 등장하면 그대로 둔다** |
| 공덕 / 공덕 쌓기 | (대체어 없음 — 문맥에 맞게 재작성) | "공덕", "공덕을 쌓다" 등 불교적 함의의 표현 일괄 금지. "선한 일", "선행", "여호와께서 기뻐하시는 일" 등으로 풀어 쓴다 |

> 종결 어미의 청자 존대('-습니다/-입니다')는 그대로 유지한다. 금지 대상은 **호칭에 직접 붙이는 '님/분'** 과 **주어 형제/자매에 대한 주체높임 '-시-'** 다.

## 검토 절차

요약·정리 문장을 작성한 직후, 내가 쓴 부분에 한해 자가 검사한다.

1. **원문/내문장 구분** — 출력에서 인용·추출한 원문과 내가 쓴 요약·안내 문장을 구분한다. 원문은 건드리지 않는다.
2. **문자열 검색** — 내가 쓴 문장에서 "통치체", "제명", "형제님", "자매님", "장로님", "장로 형제", "사역", "목회", "교회", "간증", "예수님", "공덕" 키워드를 검색한다.
3. **문맥 확인** — "형제"·"자매" 뒤에 "께서/하셨/주셨" 등 주체높임 결합이 있는지 확인한다.
4. **치환·재작성** — 위 표에 따라 교정한다. "목회"는 단순 치환이 어려우므로 문장을 재작성한다.
5. **"교회"·"장로" 문맥 판단** — "교회"는 타종교 지칭이 아니면 "회중"으로, "장로 형제"는 직책 지칭이 필수가 아니면 "형제"로 축소한다.
6. **재독** — 교정 후 어미·조사 불일치가 없는지 한 번 더 읽는다.

## 적용 범위

| 명령어 | 적용 정도 |
|---|---|
| `/meeting-workbook` | **적용** — 집회 순서 요약·표 설명·안내 문구 등 내가 쓰는 문장 |
| `/wt-preview` | **적용** — 예습자료의 정리·안내 문구 등 내가 쓰는 문장 |
| `/lookup-verse` | 원문 출력 — 거의 불필요 (덧붙이는 설명이 있을 때만) |
| `/lookup-study` | 원문 출력 — 거의 불필요 |
| `/read-jwpub` | 원문 출력 — 거의 불필요 |
| `/extract-jwb-subtitle` | 자막 원문 보존 — 불필요 |
